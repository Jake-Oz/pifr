# PIFR-070

## Objective

MOS Schedule 3 Section 2.2 Unit 2.2.1 15.1.1: Describe the instrument indications that would indicate the following: (a) scalloping; (b) VOR station passage; (c) abeam VOR station; (d) VOR radial the aircraft is on; (e) track error and/or drift experienced.
