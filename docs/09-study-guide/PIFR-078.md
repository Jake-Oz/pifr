# PIFR-078

## Objective

MOS Schedule 3 Section 2.2 Unit 2.2.1 15.2.3: Describe how the following are determined using an ADF relative bearing indication: (a) NDB station passage; (b) abeam NDB station; (c) NDB bearing the aircraft is on; (d) track error and/or drift experienced.
