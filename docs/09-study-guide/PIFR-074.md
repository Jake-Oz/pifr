# PIFR-074

## Objective

MOS Schedule 3 Section 2.2 Unit 2.2.1 15.1.5: Fix position, given cockpit instrument indications utilising 2 VOR stations.

## Why it matters

Two-VOR position fixing connects the official positive-fix rule with cockpit plotting technique; only the positive-fix rule is currently verified.
