# PIFR-072

## Objective

MOS Schedule 3 Section 2.2 Unit 2.2.1 15.1.3: State VOR omni-bearing selector (OBS) settings required to provide command indications when flying on given tracks both to and from the VOR.

## Why it matters

OBS command indications are practical VOR technique material, but the current verified sources do not explain the to/from setting method.
